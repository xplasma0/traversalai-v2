# `traversalai clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `traversalai clawbot qr` (same behavior as [`traversalai qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `traversalai clawbot qr` -> `traversalai qr`
