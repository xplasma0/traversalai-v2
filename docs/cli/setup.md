# `traversalai setup`

Initialize `~/.traversalai/traversalai.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
traversalai setup
traversalai setup --workspace ~/.traversalai/workspace
```

To run the wizard via setup:

```bash
traversalai setup --wizard
```
