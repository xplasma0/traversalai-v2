# `traversalai tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
traversalai tui
traversalai tui --url ws://127.0.0.1:18789 --token <token>
traversalai tui --session main --deliver
```
